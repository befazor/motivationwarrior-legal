# Privacy Policy for MotivationWarrior_AutoUploader

Effective Date: 2025-01-01

MotivationWarrior_AutoUploader ("the App") is a tool used to upload videos to TikTok on behalf of the user. This Privacy Policy explains how user data is handled.

## 1. Data We Collect
The App only collects the information that TikTok’s API provides after the user grants permission:
- Basic profile information (open_id, display name, avatar)
- Video upload permissions
- Video list permissions

We do NOT collect:
- Email addresses
- Phone numbers
- Passwords
- Personal contact details
- Analytics or cookies

## 2. How We Use Data
The App uses TikTok-provided data exclusively to:
- Authenticate the user via TikTok Login Kit
- Upload videos as drafts to the user’s TikTok account
- Retrieve the list of user's public videos (if authorized)

No data is stored permanently. Data is only processed during the active session.

## 3. Data Sharing
We do NOT share, sell, or transfer user data to any third parties.

## 4. Data Storage
No personal user data is stored on any server. All data is processed temporarily through secure TikTok API connections.

## 5. Third-Party Services
The App uses TikTok Login Kit and TikTok Content Posting API.

## 6. User Rights
Users may revoke access at any time via the TikTok app.
